Tentang SSL

Apa itu SSL?

SSL adalah protokol keamanan yang memberikan perlindungan terhadap komunikasi data Anda di internet menggunakan teknologi enkripsi.

Apakah fungsi SSL?

SSL berfungsi untuk mengenkripsi / mengacak data yang terkirim dari website Anda ke pengunjung dan sebaliknya. SSL memastikan data tidak dapat disadap dan diketahui oleh pihak lain.

Mengapa Anda membutuhkan SSL?

Browser seperti Google Chrome akan memunculkan peringatan NOT SECURE yang akan menurunkan tingkat kepercayaan pengunjung website. Jika website Anda menggunakan SSL, alamat URL Anda akan menjadi berawalan https dan menampilkan logo gembok, pertanda website Anda aman dan terpercaya sehingga semakin menambah rasa aman pada pengunjung.

Website yang menggunakan SSL juga akan mendapatkan peringkat yang lebih baik dari Google sehingga dapat muncul di peringkat atas dari hasil pencarian.

Tipe dan Jenis SSL

Domain Validation (DV SSL)

Jika website Anda adalah website personal, blog, atau website usaha yang baru dirintis, maka DV (Domain Validated) SSL sudah cukup utuk Anda.

Sertifikat DV SSL akan mengenkripsi traffic dari dan ke website Anda namun tidak ada validasi lebih lanjut. Artinya traffic website Anda aman namun belum bisa dikatakan “terpercaya”.

Organization Validation (OV SSL)

Organization Validation SSL dibutuhkan oleh website bisnis atau jenis webiste yang menerima input data dari pengunjung website. OV SSL memastikan website yang diakses pengunjung adalah benar-benar website yang valid dan aman sehingga meminimalkan resiko phising.

Jika website Anda menerima pendaftaran user, memiliki login untuk admin/user, menyimpan password, dll, maka setidaknya gunakan OV SSL.

Extended Validation (EV SSL)

Extended Validation SSL disebut juga green bar SSL meningkatkan kepercayaan karena akan menampilkan padlock dan baru hijau di browser.

Tidak hanya aman, EV SSL memberikan kepercayaan tertinggi dan mencegah phising terhadap website Anda. Itu mengapa type sertifikat ini menjadi keharusan bagi website enterprise, banking, dan e-commerce.

Benefit SSL untuk Website Anda

Melindungi Transaksi Data

SSL adalah protokol keamanan yang memberikan perlindungan terhadap komunikasi data Anda di internet menggunakan teknologi enkripsi.

Lebih dipercaya Pengunjung

Pengunjung lebih percaya saat men-submit data dan informasi mereka ke website Anda karena proses pengiriman datanya telah terenkripsi sehingga tidak dapat dibajak oleh pihak lain.

Meningkatkan Rangking di Google

Google memberikan ranking yang lebih tinggi untuk website yang dilengkapi SSL. Website dg SSL jenis OV dan EV bahkan diranking akan lebih tinggi lagi.

Mencegah Phishing

Anda dapat mencegah website dari resiko phising dengan SSL Bisnis dan SSL Enterprise. Penerbit SSL akan memastikan bahwa website yang dilindungi dengan SSL adalah benar-benar milik perusahaan yang legitimate melalui proses vetting menggunakan dokumen legal perusahaan.

Site Seal

Site Seal adalah image (biasanya berbentuk badge) yang menandakan website Anda terenkripsi dan aman. Pengunjung akan merasa lebih terlindungi dan percaya diri dalam menjelajah website dan mengirimkan data melalui website Anda.

Bebas Warning dari Chrome

Dengan menggunakan SSL, browser Chrome tidak akan lagi memberikan peringatan bagi pengunjung bahwa website Anda tidak aman. Pengunjungpun merasa lebih konfiden.

FAQ

1. Apakah itu Sertifikat SSL?

SSL atau Secure Socket Layer adalah protokol keamanan yang memberikan perlindungan terhadap komunikasi data Anda di internet menggunakan teknologi enkripsi.

Selain itu SSL juga memuat informasi penting mengenai website yang dikunjungi. Pada Organization Validation SSL (OV SSL) dan Extended Validation SSL (EV SSL), data tentang pemilik website dimuat pada sertifikat sehingga pengunjung dapat melihat siapa sebenarnya pemilik website yang sedang dikunjunginya, memastikan pengunjung tidak terkena phising.

2. Bagaimanakah Cara Kerja SSL?

Secara singkat SSL mengamankan transmisi data antara pengunjung dan website. SSL mengacak data yang keluar masuk tersebut menggunakan teknologi enkripsi sehingga tidak dapat dibajak oleh pihak yang tidak berkepentingan.

Contoh mudahnya adalah saat user mendaftar atau login pada website Anda dan memasukkan username dan password. Tanpa SSL, username dan passwordnya akan dikirimkan dalam bentuk text tanpa acak, sedangkan dengan SSL username dan passwordnya akan diacak sehingga aman. Itu kenapa semua website internet banking, e-commerce, website yang menerima pendaftaran dan login, dan website-website lainnya wajib menggunakan SSL demi keamanan website dan pengunjungnya.
